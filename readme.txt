Plugin de gerenciamento de imóveis para imobiliárias.

=== Recursos ===

- Cadastro de Imóveis
- Filtros personalizados
- Cadastro e gerenciamento de proprietários
- Sistema CRM integrado
- Integração com portais

=== Portais integrados ===

- VivaReal
- Imovel WEB
- Wikihome
